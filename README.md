# nadeemahmed.co.uk

Personal website and portfolio for Nadeem Ahmed.

Live site: [nadeemahmed.co.uk](https://nadeemahmed.co.uk)

## About

This is a minimal, fast-loading personal site built to showcase my work and background in product design, UX, and quality engineering.

## Tech

- HTML
- CSS
- Hosted on Netlify

## Run locally

No build step needed. Open `index.html` directly in your browser, or run a simple local server:

```bash
python3 -m http.server 8080
```

Then visit `http://localhost:8080`.

## Deploy

This site is deployed via Netlify from the GitHub repo.
